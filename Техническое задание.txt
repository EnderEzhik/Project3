Локально запускаемый сайт для управления настройками и функционалом программы.

На главной странице:
- Лого-текст
- Кнопка запуска программы
- Кнопка закрытия программы
- Кнопка фильтры
- Кнопка чаты

Кнопка запуска программы: 
- Запускает основную программу

Кнопка закрытия программы: 
- Закрывает основную программу

Кнопка фильтры: 
- Открывает страницу с фильтрами для работы основной программы
  - Черный и белый спискок пользователей

Кнопка чаты: 
- Открывает страницу с списком чатов

Страница с фильтрами:
- Слева сверху кнопка возвращающая на главную страницу
- Слева по центру черный список пользователей
- Справа по центру белый список пользователей
- Под каждым из списков кнопка добавления нового элемента в список, при нажатии на которую открывается страница с списком всех доступных для добавления элементов (элементов, которых нет ни в черном, ни в белом списке)

Страница с доступными элементами:
- Слева сверху кнопка возвращающая на главную страницу
- По центру список всех доступных элементов
- Слева каждого элемента кнопка добавления его в список (в какой список - смотря из какого списка перешли на эту страницу)

Страница с чатами:
- Список всех доступных чатов
- При нажатии на чат, открывается страница с историей выбранного чата
